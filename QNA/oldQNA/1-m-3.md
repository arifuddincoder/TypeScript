# ЁЯЫая╕П How to Write a TypeScript Program? (With Bengali Guide)

<details>
<summary>тЭУ <strong>Question: How do I write a TypeScript program?</strong></summary>

This guide explains step-by-step how to write, compile, and run a TypeScript program from scratch using the TypeScript Compiler (`tsc`).

</details>

---

<details>
<summary>ЁЯУЭ <strong>Step-by-step: Write and Run a TypeScript Program (English)</strong></summary>

### тЬЕ 1. Install TypeScript Compiler

```bash
npm install -g typescript
```

> This installs the global `tsc` (TypeScript Compiler) command.

---

### тЬЕ 2. Create a TypeScript File

Create a file named `app.ts`:

```ts
// app.ts
function greet(name: string): string {
  return `Hello, ${name}!`;
}

console.log(greet("Arif"));
```

---

### тЬЕ 3. Compile TypeScript to JavaScript

Use the `tsc` command:

```bash
tsc app.ts
```

It will generate a file: `app.js`

---

### тЬЕ 4. Run the JavaScript File

```bash
node app.js
```

You will see output:  
```
Hello, Arif!
```

---

### тЬЕ 5. (Optional) Initialize tsconfig.json

If you're building a project:

```bash
tsc --init
```

It creates a `tsconfig.json` file with TypeScript configuration.
